# See More - Example App

This example demonstrates all features of the `see_more` package.

## Running the Example

```bash
cd example
flutter run
```

## Examples Included

1. **Basic Character Trimming** - Default usage with character count
2. **Line-based Trimming** - Trim by number of lines
3. **Fade Effect** - Gradient fade like Instagram/Twitter
4. **Custom Styling** - Custom colors, fonts, and text
5. **Callbacks** - Track expand/collapse events
6. **Initially Expanded** - Start in expanded state
7. **Custom Ellipsis** - Different ellipsis text
8. **Center Aligned** - Text alignment options
9. **Dark Background** - Fade effect on dark theme
10. **Short Text** - No trimming needed
11. **Custom Animation** - Slow animation with elastic curve
12. **2 Lines Fade** - Minimal line count with fade
13. **No Word Boundary** - Cut mid-word (disabled)
14. **Social Media Style** - Instagram/Twitter-like post
